About ORB

ORB was my attempt at my own forum software.

I always thought forum software that was a combination of the linear-forum style like phpBB, XenForo, MyBB etc. and the Reddit post system would be neat to see. Unfortunately I only got as far as a basic HTML & CSS demo up.
